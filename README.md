# Samples Repository for Antipode Components.

This application is intended to be used for testing purposes.

# Initialization

1. git clone https://github.com/antipodejs/samples.git

2. npm install

3. mkdir lib

4. cd lib

5. git clone https://github.com/antipodejs/antipode.git

6. cd ../..

7. npm start

8. open http://localhost:8080
